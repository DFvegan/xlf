秦朝(前221年~前207年)是由战国时期的秦国发展起来的中国历史上第一个统一的封建王朝， 秦人的祖先大费是黄帝之孙颛顼的后裔 ，舜赐其嬴姓。秦穆公时，任贤使能，虚心纳谏，灭国十二，开地千里，国力日盛。前361年，秦孝公继位，重用商鞅两次变法，使秦国的经济得到发展，军队战斗力不断加强，发展成为战国后期最富强的诸侯国。
